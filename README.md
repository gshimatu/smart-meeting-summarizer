# SmartMeeting Summarizer

SmartMeeting Summarizer est un projet open-source qui transforme automatiquement des réunions (audio ou texte) en informations exploitables :

- Résumé clair et structuré
- Décisions prises
- Actions à réaliser

Le projet est conçu comme un **produit professionnel**, évolutif vers un **SaaS**.

---

## Problème résolu

Les réunions sont souvent :
- longues
- mal documentées
- peu exploitables

SmartMeeting Summarizer permet de passer directement de la réunion brute à un **plan d’action clair**.

---

## Fonctionnalités principales (MVP)

- Upload de fichiers audio (.mp3, .wav) ou texte (.txt)
- Transcription automatique (Speech-to-Text)
- Analyse intelligente :
  - Résumé
  - Décisions
  - Actions
- Export des résultats (Markdown)

---

## Technologies utilisées

- Python
- FastAPI
- Whisper (Speech-to-Text)
- NLP / LLM
- Streamlit (frontend MVP)
